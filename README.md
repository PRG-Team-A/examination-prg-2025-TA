# Customer self-checkout system

This is a self-checkout system similar to the ones in supermarkets.

## Running from sources

1. Make sure you have the Java Development Environment for Java 25 installed.
2. To run the development server, navigate to the projects root directory and run:

```bash
mvn spring-boot:run
```

3. The development server should be running (ideally)
